## Item 3 — Filtros do Ecossistema viram aba que recolhe

**Screenshot:** Image 3 — comentário "Aba que recolhe"

**Onde:** `src/pages/Ecossistema.jsx` linhas 153-168 (bloco `.filter-section` + `.solution-filters`).

**Problema atual:** 6 botões de filtro empilham vertical no mobile, ocupando muito da viewport antes do usuário ver qualquer solução. No screenshot dá pra ver a lista expandida cobrindo metade da tela.

**Proposta:**
- Desktop (>= 1024px): mantém comportamento atual de chips horizontais.
- Tablet/mobile (< 1024px): vira **acordeão colapsável**.
  - Header padrão: mostra o filtro **ativo** (ex.: "🛡️ Quero Proteger o Negócio") + chevron pra abrir.
  - Ao tocar, expande a lista dos 6 filtros.
  - Ao escolher um, fecha automaticamente.
  - Estado padrão: fechado, mostrando "Todas as Soluções".

**Implementação:**
- Adicionar `useState(false)` pra controlar `filtersOpen`.
- Wrapper `.solution-filters` ganha classe condicional `is-open` / `is-closed`.
- Botão header novo só aparece no breakpoint mobile (CSS `@media`).
- Animação CSS de altura (max-height 0 → 600px com transition).
- Acessibilidade: `aria-expanded`, `aria-controls`.

## Item 4 — "O ponto central desse exercício" quebrado no mobile

**Screenshot:** Image 5 — comentário "Quebrado"

**Onde:** `src/pages/ConsultorioEmpresarial.jsx` linhas 180-196.

**Bug raiz:** o grid usa estilos **inline** com `gridTemplateColumns: 'repeat(3, 1fr)'`, que não respondem a media queries. No mobile, as 3 colunas continuam coladas e o texto fica espremido em colunas estreitas (cada card vira ~33vw com 5-7 palavras esmagadas).

```js
// ATUAL (linha 182):
<div style={{ display: 'grid', gridTemplateColumns: 'repeat(3, 1fr)', gap: '1rem', marginBottom: '1.5rem' }}>
```

**Proposta:** mover layout pra classe CSS responsiva e remover inline styles de grid (alinhado com a regra de `CLAUDE.md`: "All layout styles (grid, flex) are in CSS classes, NOT inline styles. Inline `style={}` on layout elements prevents media query overrides.").

**JSX novo:**
```jsx
<div className="ce-ponto-central">
  <div className="ce-ponto-central-cards">
    <div className="ce-ponto-card"><p>A maioria dos empresários empreende sozinho por muito tempo</p></div>
    <div className="ce-ponto-card"><p>Decide sem parâmetro</p></div>
    <div className="ce-ponto-card"><p>Aprende com os erros das escolhas ruins</p></div>
  </div>
  <p className="ce-ponto-central-footer">Isso não é incompetência — é falta de apoio estruturado. É sinal de uma trajetória longa em um ambiente difícil e hostil.</p>
</div>
```

**CSS novo (em `src/index.css`):**
```css
.ce-ponto-central { padding: 2.5rem; background: var(--cinza-escuro); border-radius: var(--border-radius-lg); border: 1px solid rgba(247,238,235,0.08); }
.ce-ponto-central-cards { display: grid; grid-template-columns: repeat(3, 1fr); gap: 1rem; margin-bottom: 1.5rem; }
.ce-ponto-card { background: var(--cobre); border-radius: var(--border-radius-lg); padding: 1.5rem; }
.ce-ponto-card p { color: var(--off-white); margin: 0; font-size: 0.95rem; }

@media (max-width: 768px) {
  .ce-ponto-central-cards { grid-template-columns: 1fr; }
}
```
